Labels API

List all Labels for this Repository

GET /repos/:user/:repo/labels.json

Response

[{"url"=>"https://api.github.com/repos/octocat/Hello-World/labels/bug.json",
  "name"=>"bug",
  "color"=>"f29513"}]

Create a Label

POST /repos/:user/:repo/labels.json

Input

{
  name: "String",
  color: "hex" // "ffffff" (6 characters, no '#')
}

Response

{"url"=>"https://api.github.com/repos/octocat/Hello-World/labels/bug.json",
 "name"=>"bug",
 "color"=>"f29513"}

Get a single Label

GET /repos/:user/:repo/labels/:id.json

Response

{"url"=>"https://api.github.com/repos/octocat/Hello-World/labels/bug.json",
 "name"=>"bug",
 "color"=>"f29513"}

Update a Label

PATCH /repos/:user/:repo/labels/:id.json

Input

{
  name: "String",
  color: "hex" // "ffffff" (6 characters, no '#')
}

Response

{"url"=>"https://api.github.com/repos/octocat/Hello-World/labels/bug.json",
 "name"=>"bug",
 "color"=>"f29513"}

Delete a label

DELETE /repos/:user/:repo/labels/:id.json

Response

{}
