Breaking BETA Changes

We’re making some small tweaks to the API during the BETA phase. Old behavior is still supported, but due to be removed by June 1. Please be sure to update your app in time.

  • Update requests used to accept the PUT verb. Now POST or PATCH should be used.
  • All URLs had a .json extension. They don’t anymore (but old requests work still).
